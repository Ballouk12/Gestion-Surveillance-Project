# 📝 Système de Gestion des Examens

Une application web complète pour la gestion des examens utilisant **Vite.js (React)** pour le frontend et **Spring Boot** pour le backend.

---

## 🚀 Technologies Utilisées

### Backend
- ☕ **Java JDK 17**
- 🖥️ **Spring Boot 3.x**
- 🗄️ **MySQL 8.0**
- ⚙️ **Maven**

### Frontend
- ⚡ **Vite.js + React**
- 🎨 **Material Tailwind**
- 🛠️ **Heroicons**
- 🔀 **React Router DOM**

---
